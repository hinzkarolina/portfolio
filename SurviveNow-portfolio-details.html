SurviveNow

The survival game project was completed as part of my master's thesis. The game was implemented using the Unity engine and the C# programming language. Adobe Illustrator was used to create 2D graphics and some textures.

Henderson Island was the inspiration for the "SurviveNow" game design. The "SurviveNow" game offers a dynamic survival experience on the island. The player's main task is to repair the helicopter however, he also has to make sure that he survives on the island.
